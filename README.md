☀️ Yaz Okulu Platformu (Summer School Platform)
Bu proje, farklı üniversitelerin yaz okulu derslerini tek bir merkezde toplayan, öğrencilerin ders arama ve şehir analizi süreçlerini kolaylaştıran kapsamlı bir platformdur. Platform, veri madenciliği ve yapay zeka tekniklerini kullanarak öğrencilere verimli bir tercih süreci sunar.

📋 Proje Özeti
Öğrencilerin farklı üniversite web sitelerini tek tek inceleme zorunluluğunu ortadan kaldıran sistem, tüm yaz okulu duyurularını merkezi bir veritabanında toplar. Sistem, ders detaylarının yanı sıra yapay zeka desteğiyle şehirlerin yaşam koşullarını da analiz eder.

✨ Öne Çıkan Özellikler
Merkezi Ders Havuzu: Üniversite web sitelerinden toplanan güncel ders listeleri.

Gelişmiş Filtreleme: Kredi, AKTS, kontenjan ve ders programına göre detaylı arama seçenekleri.

AI Şehir Analizi: Seçilen üniversitenin bulunduğu şehrin yaşam maliyeti, ulaşım ve sosyal imkanlarının artı-eksi analizi.

Çapraz Platform Desteği: Hem Web (React) hem de Mobil (React Native) üzerinden erişim imkanı.

Otomatik Veri Toplama: BeautifulSoup ve Scrapy ile üniversite duyurularının düzenli olarak çekilmesi.

🛠️ Teknoloji Yığını
Backend: ASP.NET Core (RESTful API)

Frontend: React (Web) & React Native (Mobile)

Database: PostgreSQL

Web Scraping: BeautifulSoup & Scrapy

AI / Machine Learning: Pandas, Scikit-learn, Flask/FastAPI

Infrastructure: AWS (EC2, S3, RDS), Docker 

Security: JWT (JSON Web Token) tabanlı kimlik doğrulama

🏗️ English Version
Project Goal
A centralized platform designed to aggregate summer school courses from various universities, allowing students to search, filter, and analyze options efficiently.

The Problem Solved
Navigating multiple university websites for summer school announcements is time-consuming. This project provides a unified search engine for course credits, ECTS, and schedules.

AI-Powered Insights
The system uses a machine learning model to provide a comparative analysis of the cities where universities are located, covering living costs, transportation, and social amenities.

Technical Implementation
Backend: Developed using ASP.NET Core for scalable RESTful APIs.

AI Engine: Built with Scikit-learn to analyze city/university data and provide recommendations.

Data Acquisition: Automated web scraping tools integrated to pull real-time data from official university portals.
